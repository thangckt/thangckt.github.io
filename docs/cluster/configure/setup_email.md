# Check outlook mail from gmail

## 1. Import email

use one of the following methods to import email from outlook to gmail

### Import email from outlook (POP setting)

Gmail -> Settings -> Accounts and Import -> Import mail and contacts -> Import mail and contacts

- Step 1: Sign into your other email account
  - Email password
  - POP username: your email address
  - POP server: e.g., outlook.office365.com
    - Port: 995 SSL: checked
- Step 2: Import options
  - Import mail
  - Leave a copy of messages on the server: checked
  - Add label: checked
