[Visual Studio Code](https://code.visualstudio.com/) is free, lightweight, and platform-independent.

## Why should use VScode for Jupyter Notebook

- A single editor for many languages
- Supports multiple python file types
- Execution time
- Outline mode
- Jupyter variable explorer
- Data viewer
- Code formatting
- Cell debugging
- Split editors
- Git integration
- Better diffs (may not be available in Jupyter)
- Intellisense)
- Variable peeking
- Gather code
- Snippets
- Connect to a remote server
- Supports WSL
- Viewing plots
- Plugins

read more [here](https://pbpython.com/vscode-notebooks.html)
